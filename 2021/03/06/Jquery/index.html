jQuery 介绍

什么是jQuery ？

jQuery是一个快速、简洁的JavaScript框架，是继Prototype之后又一个优秀的JavaScript代码库（或JavaScript框架），极大地简化了 JavaScript 编程。

jQuery 的核心理念

write less, do more（写的 更少，做的更多）。

jQuery的核心函数$()

$(function(){})就相当于js中的windows.onload=function(){}，极大的简化了操作，后面我们会大量用到$()核心函数。
